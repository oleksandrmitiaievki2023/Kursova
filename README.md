# Kursova
Опис програми

Програмне забезпечення для автоматичного збору, збереження та аналізу лог-файлів різних програм і процесів, що виконуються на комп’ютері.
Система дозволяє централізовано контролювати події, виявляти помилки, аналізувати активність процесів та формувати звіти.

Програма працює у фоновому режимі та:

Відстежує вказані директорії або процеси.
Зчитує нові записи логів у реальному часі.
Парсить та нормалізує дані.
Зберігає їх у структурованому вигляді.
Виконує аналіз та формує звіти.
Генерує сповіщення у разі критичних подій.
Основні функції та фічі

1. Моніторинг логів у реальному часі

Автоматичне відстеження змін у лог-файлах без перезапуску програми.

2. Підтримка різних форматів

Обробка TXT, LOG, JSON, CSV та інших форматів.

3. Нормалізація даних

Приведення різних типів логів до єдиного внутрішнього формату (timestamp, рівень, джерело, повідомлення).

4. Розширена фільтрація

Фільтрація за датою, рівнем (INFO/WARNING/ERROR), процесом або ключовими словами.

5. Повнотекстовий пошук

Швидкий пошук по всій базі логів із підтримкою регулярних виразів.

6. Виявлення повторюваних помилок

Автоматичне визначення помилок, що часто повторюються.

7. Статистичний аналіз

Формування статистики за періодами (день, тиждень, місяць).

8. Система сповіщень

Повідомлення користувача про критичні події.

9. Архівація та ротація логів

Автоматичне стискання та очищення старих записів.

10. Генерація звітів

Експорт звітів у форматах CSV або PDF для подальшого аналізу.
